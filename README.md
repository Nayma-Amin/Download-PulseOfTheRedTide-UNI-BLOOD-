# UNI-BLOOD - Pulse of the Red Tide - App

UNI-BLOOD (Pulse of the Red Tide) is a Blood Donation Docial Media App. This app was designed and implemented to bridge the gap between doners and recipeints.
We often struggle to find blood doners when it's urgent, we dug through social medias where people share their desired content and less or urgency. Making the search more time consuming than necessary.
That's where our app comes in handy. This app allows users to find doners within their area, and also the nearest possible avaible doners. The in app messaging makes it easier to contact and shsare information at the users consent.
This app was designed to encourage young generation to step ahead and contribute to humanity without a struggle.

---

## Features

- User sign up and login
- Users makes blood donation posts, events.
- User can interact with doners and recipients. 
- In app messaging to avoid extra user cost for communication using built in message or calls.
- Emergency notification alerts helps bringing doners and recipinets in touch easier.
- Users can look for nearby doners, help centers and hospitals.
- User can update and manage their profile and data.
- User receives stars for donation, making them the symbols of humanity.
- User can track eligibility with a built in tracker.
- User can check doner compatibility.
- Admin dashboard maintains authenticity
